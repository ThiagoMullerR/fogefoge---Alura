## Como rodar

1. Pelo Terminal, compile o jogo:
    ```bash
    gcc fogefoge.c mapa.c ui.c -o fogefoge
    ```

2. Execute o jogo compilado.
    ```bash
    ./fogefoge
    ```

## Como jogar

- **W**: Move para cima.
- **S**: Move para baixo.
- **D**: Move para direita.
- **A**: Move para esquerda.
- **B**: Explode a bomba.
